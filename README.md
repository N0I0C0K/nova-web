# Nova Web

这是 Nova 独立游戏研究社团的网站代码

## 技术栈

- next.js - page router
- mongodb
- prisma
- next-auth
- react-markdown
- tailwind-css
- chakra-ui

## 页面

- 首页
  - 活动展示
  - 作品展示
  - 小组展示
- 试玩页
- 文章
  - 个人写的文章
  - 可以评论

考虑加入的功能：

- 个人页面
